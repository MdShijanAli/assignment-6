Difference of var, let, const

Var:

The var statement declares a variable. Variables are containers for storing information. Creating a variable in JavaScript is called "declaring" a variable: var carName; After the declaration, the variable is empty (it has no value).

Let:

let allows you to declare variables that are limited to the scope of a block statement, or expression on which it is used, unlike the var keyword, which declares a variable globally, or locally to an entire function regardless of block scope

Const:

Constants are block-scoped, much like variables declared using the let keyword. The value of a constant can't be changed through reassignment (i.e. by using the assignment operator), and it can't be redeclared (i.e. through a variable declaration).

Difference of arrow function and regular function

Arrow Function:

Arrow functions are a new way to write anonymous function expressions, and are similar to lambda functions in some other programming languages, such as Python. Arrow functions differ from traditional functions in a number of ways, including the way their scope is determined and how their syntax is expressed.

Regula Function:

In mathematics, a regular function in the sense of algebraic geometry is an everywhere-defined, polynomial function on an algebraic variety V with values in the field K over which V is defined.

Difference of map,foreach,filter,find

Map:

map() creates a new array from calling a function for every array element. map() calls a function once for each element in an array. map() does not execute the function for empty elements. map() does not change the original array. example: const numbers = [3, 4, 5, 2]; const resultMap = numbers.map(num => num + 50); console.log(resultMap); output: [ 53, 54, 55, 52 ]

FoeEach:

forEach is a JavaScript Array method. It is used to execute a function on each item in an array. Lists, sets, and all other list-like objects support the forEach method. example: const numbers = [3, 4, 5, 2]; numbers.forEach(number => { console.log(number); }) output: 3 4 5 2

Filter:

The filter() method creates a new array filled with elements that pass a test provided by a function. The filter() method does not execute the function for empty elements. The filter() method does not change the original array. Example: const numbers = [3, 4, 5, 2]; const result = numbers.filter(num => num < 5) console.log(result); Output: [ 3, 4, 2 ]

Find:

The find() method returns the value of the first element that passes a test. The find() method executes a function for each array element. The find() method returns undefined if no elements are found. The find() method does not execute the function for empty elements.
Example: const numbers = [3, 6, 5, 2]; const result = numbers.find(num => num > 3); console.log(result); Output: 6

Why use template string

Template String
